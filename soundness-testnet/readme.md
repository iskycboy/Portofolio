# Soundie PFP Maker 🐬

**Soundie PFP Maker** is a web-based tool specifically designed for the **Soundness** community. It allows users to easily personalize their Profile Pictures (PFPs) with a variety of unique assets and accessories.

## ✨ Key Features
* **Seamless Customization**: Quickly select and mix assets like hats, glasses, and outfits.
* **Community-Centric Assets**: Curated collection tailored specifically for the *Soundies* identity.
* **Lightweight & Fast**: Optimized for a smooth and responsive user experience.

## 🚀 Tech Stack
* **Framework**: Next.js / React
* **Styling**: Tailwind CSS
* **Deployment**: Vercel

## 🛠️ Local Setup
1. Clone the repository:
   git clone https://github.com/nurdin/soundie-pfp-maker.git

2. Navigate to the project directory:
   cd soundie-pfp-maker

3. Install dependencies:
   npm install

4. Run the development server:
   npm run dev
